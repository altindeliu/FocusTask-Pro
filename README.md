# FocusTask Pro

A React Native task management application built with TypeScript.

# Demo video

https://drive.google.com/file/d/16C8kBcfq9kLqt1zuz_K8KR-OccK9CGXR/view?usp=sharing

## Features

- Add Task
- View Task Details
- Complete Tasks
- Restore Tasks
- Delete Tasks
- Search Tasks
- Filter Tasks (All / Pending / Completed)
- Local Storage with AsyncStorage
- Motivational Quotes

## Technologies

- React Native
- TypeScript
- React Navigation
- AsyncStorage

## Installation

```bash
npm install
```

## Run the App

```bash
npx expo start
```
